## 📖 Overview

Python Port Scanner is a Python-based networking and cybersecurity project created to learn:

* Socket Programming
* TCP/IP Networking
* Python Automation
* Network Reconnaissance Fundamentals

The tool scans a target host and identifies open TCP ports in a controlled and authorized environment.

---

## 🚀 Installation

```bash
git clone https://github.com/adityainfosec/Python-Port-Scanner.git

cd Python-Port-Scanner

python3 port_scanner.py
```
