## 📖 Overview

Python Port Scanner is a Python-based cybersecurity project developed to understand TCP/IP networking, socket programming, and network reconnaissance. The tool scans a target IP address, checks commonly used TCP ports, identifies open ports, and helps users understand how network services communicate over a network.

This project is intended for educational purposes and authorized security testing in lab environments.
